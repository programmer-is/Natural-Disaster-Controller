# 📖 درباره پروژه

```text
┌─────────────────────────────────────────────────────────────┐
│  بلایای طبیعی همیشه قابل پیش‌بینی نیستند، اما آمادگی          │

└─────────────────────────────────────────────────────────────┘
```

**Natural Disaster Controller** یک سامانه تحت وب برای مدیریت، پایش و تحلیل اطلاعات مربوط به بلایای طبیعی است.

این سامانه بستری یکپارچه برای:

✅ ثبت و مدیریت هشدارها

✅ دریافت گزارش‌های مردمی

✅ پایش وضعیت مناطق

✅ تحلیل اطلاعات بحران

✅ افزایش آمادگی عمومی

فراهم می‌کند.

# 🚀 راه‌اندازی سریع

## ① نصب WampServer

```text
✔ Apache
✔ PHP
✔ MySQL
```

پس از اجرا، آیکون WampServer باید سبز رنگ باشد.

---

## ② انتقال پروژه

```text
C:\wamp64\www\
```

مثال:

```text
C:\wamp64\www\Natural Disaster Controller
```

---

## ③ ساخت دیتابیس

وارد شوید:

```text
http://localhost/phpmyadmin
```

دیتابیس زیر را ایجاد کنید:

```text
natural_disaster_db
```

---

## ④ وارد کردن فایل SQL

```text
Database
   │
   ├── Import
   │
   └── Select SQL File
```

فایل SQL موجود در پروژه را وارد نمایید.

---

## ⑤ بررسی اتصال دیتابیس

فایل:

```text
config/database.php
```

تنظیمات:

```php
$host = "localhost";
$dbname = "natural_disaster_db";
$username = "root";
$password = "";
```
